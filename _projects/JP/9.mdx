# 画像掲示板アプリ


私が作成したプロジェクトの一つは、**AWS**、**Node.js**、**React**を使用した画像掲示板アプリです。<br/>

<br/>

アプリの核心機能はシンプルです。  
ユーザーは画像をアップロードし、他の人の画像を確認することができ、コメントや「いいね」機能も含まれています。<br/>

<br/>

**Googleログイン**機能も実装されており、Googleアカウントを通じて簡単にログインして使用することができます。<br/>

<br/>

バックエンドはNode.jsでAPIを提供し、画像ファイルはAWS S3に保存されるように設計しました。  
フロントエンドはReactを使用してユーザーインターフェースを実装しました。<br/>

<br/>

現在はサーバーを停止しています。<br/>
簡単なプロジェクトですが、AWSとReact、Node.jsを連携させ、Google認証まで適用してみた経験があるアプリです。<br/>
