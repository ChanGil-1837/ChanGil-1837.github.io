# 이미지 게시판 앱


제가 만든 프로젝트 중 하나는 <b>AWS</b>, <b>Node.js</b>, <b>React</b>를 사용한 이미지 게시판 앱입니다.<br/>

<br/>

앱의 핵심 기능은 간단합니다.  
사용자들은 이미지를 업로드하고, 다른 사람의 이미지를 확인할 수 있으며, 댓글과 좋아요 기능도 포함되어 있습니다.<br/>

<br/>

<b>구글 로그인</b> 기능도 구현되어 있어, 구글 계정을 통해 간편하게 로그인하고 사용할 수 있습니다.<br/>

<br/>

백엔드는 Node.js로 API를 제공하고, 이미지 파일은 AWS S3에 저장되도록 설계했습니다.  
프론트엔드는 React를 사용하여 사용자 인터페이스를 구현했습니다.<br/>

<br/>

지금은 서버를 내려둔 상태입니다.<br/>
간단한 프로젝트지만, AWS와 React, Node.js를 연동하고 구글 인증까지 적용해본 경험이 있는 앱입니다.<br/>
